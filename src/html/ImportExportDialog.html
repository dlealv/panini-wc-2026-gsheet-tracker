Export current sticker counts from the COUNTRIES and COUNTS named ranges in the Stickers tab. Load a text/CSV file or paste its content. Data will be written to the COUNTS named range in the Stickers tab, preserving formatting.
1) Upload file
When a file is selected, its content will be loaded into the text area below.
2) Or paste content
Input format
Format Example Note
Format 1 MEX,1,3-7,10(2) Country code once; comma-separated stickers, ranges (3-7), repeats (10(2))
Format 2 MEX-1,MEX-5(2),MEX3 Country prefix per sticker; dash is optional (as in MEX3)
Exclusion <>MEX,1,2,3 Imports all stickers except those listed; use != or ^ instead of <>
3) Loading mode
  • Import data: clears all values in COUNTS, then loads the input.
  • Update counts clearing country counts: clears only the rows for countries in the input, then loads those countries.
  • Update counts: overwrites only the sticker positions explicitly provided; all other values remain unchanged.
Actions
  • Validate / Preview: checks syntax and shows how values will be mapped without writing to the sheet.
  • Clear: removes the selected file, clears the pasted input, clears the preview, and resets the message.
  • Import: writes the validated values to the sheet using the selected loading mode. Validation runs automatically before writing.
  • Cancel: closes the dialog without importing anything.
📌 Tip: use Validate / Preview first to confirm how counts will be mapped before importing.
Export result
The exported content uses the same Format 1 syntax accepted by the import process. Only sticker counts greater than 0 are included.
Actions
  • Download file: downloads the exported sticker data as a local file, usually into your browser's default Downloads folder.
  • Copy: copies the exported text to the clipboard.
  • Close: closes the dialog.

Input format guide

(+info)

Common rules

• Comma , or semicolon ;.
• Repeats: N(X) or N(xX) or NxX; sticker N owned X times.
• Range: A-B is an inclusive range from A to B (A>B).
• Range Repeats: A-B(X) or A-B(xX) or A-BxX; stickers A-B owned X times.
• Non-ASCII characters (e.g. 🇲🇽 MEX,1,2) are stripped automatically.
Format 1 — country code once
Syntax: CODE,N[,N(X)][,A-B][,A-B(X)]...
Examples:
  MEX,1,3-7,10(2) → MEX,1,3,4,5,6,7,10(2)
  FWC,0,1,3-5(2) → FWC,0,1,3(2),4(2),5(2)
Format 2 — country prefix per sticker
Syntax: CODE[-]N[,CODE[-]N(X)][,CODE[-]A-B]...
Format printed on the back of each Panini sticker (e.g. MEX-10).
Examples:
  MEX-1,MEX-5(2),MEX3
  FWC-1,FWC3-5(2)
• The dash between the country code and the sticker number is optional (e.g. MEX3 and MEX-3 are identical).
Exclusion operator — import missing stickers
Prefix the line with <>, !=, or ^.
The parser imports every valid sticker position except those listed.
Examples:
  <>MEX,1,2,3 → imports stickers 4–20.
  !=FWC,1-10,12-19 → imports stickers 0 and 11.
• Repeat counts inside an exclusion line are ignored; the complement always uses count 1.
• Works with Format 1 and Format 2 tokens on the same line.
